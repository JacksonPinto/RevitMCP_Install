# RevitMCP_Install

Windows installer bundle and install guide for the Revit MCP Server.

- `revit-mcp-installer-win.zip` - automated Windows setup (.bat + configure.ps1)
- `WWT_Revit_MCP_Install_Guide.pdf` - WWT-branded step-by-step guide
- `INSTALL.md` - guide in Markdown

The extension repo (RevitMCP.extension) is private: request an access token from **jackson.pinto@wwt.com**.
